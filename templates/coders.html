built with python flask, the codebase follows these principles: - **clean architecture**: separation of concerns between routes, templates, and static assets - **minimal dependencies**: only essential packages to keep the application lightweight - **security first**: input validation, csrf protection, and secure session management - **accessibility**: semantic HTML, keyboard navigation, and screen reader support - **performance**: optimized loading, minimal JavaScript, and efficient caching the core philosophy is simple: build software that serves humans, not algorithms. every line of code exists to enhance the user's experience of discovering meaningful media.
