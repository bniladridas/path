# harper

come home before it's too late

## getting started

### requirements
- python 3.8+
- google gemini api key

### setup
1. clone: `git clone https://github.com/harpertoken/path.git`
2. install: `pip install -r requirements.txt`
3. env: `GEMINI_API_KEY=your-key`
4. run: `python app.py`
